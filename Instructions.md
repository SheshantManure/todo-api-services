# Backend Interview Task

This repository is a boilerplate for a simple Express server with MongoDB already connected.

## Task

You are required to do the following:

1. Implement basic CRUD APIs for a Todo list application.
2. Define a MongoDB schema for a Todo.
3. Test the implemented APIs.

## Scope

- No user management
- No authentication or authorization
- Keep the implementation simple and minimal

## Restrictions

- Do not use Copilot or any other AI coding assistant.
